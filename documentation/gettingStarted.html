Getting Started

Prerequisites

First of all, make sure that you have the Java Runtime Environment 8 or higher installed on your system. Downloads for the Oracle JRE 8 can be found here: http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html

You can then launch SMB Level Workshop by opening `SMBLevelWorkshop-x.y.z.jar`. If that fails, you can launch it from the terminal. 

cd path/to/folder
java -Djava.library.path=. -jar "smblevelworkshop-x.y.z.jar"

Your First Project

Assuming that you have already modeled and textured a level, as well as exported it to an OBJ file format, go ahead and click on the "Import OBJ" button and find your OBJ. You will notice that upon importing, two placeables have been added for you. One that defines the starting position, and one that defines the fallout plane.

Navigating the viewport

Hold down the middle mouse button and use the WASDQE keys to fly around your scene in first person.

Adding placeables

Being able to see a stage is nice, but what't the point if you can't add any goals, bananas or the like. Go ahead and click on a placeable to add in the top left. It will be added at the position 0, 0, 0. You can also add a placeable by duplicating an existing one using Ctrl D / Cmd D.

Selecting placeables

Selecting placeables is done by simply clicking on it in the outliner on the left. Multiple can be selected if you hold down shift.

Transforming

All transforming of placeables, is done using shortcuts. If you've used Blender before, these should be familiar; if not, they should be easy to pick up.

G Grab
R Rotate
S Scale

After pressing G, R or S, you then need to define an axis to constrain transformations to. This is also done on the keyboard.

X Constrain to the X axis
Y Constrain to the Y axis
Z Constrain to the Z axis
U Constrain to the X, Y and Z axis uniformly (Useful for scaling)

Afterwards, move your mouse left or right to transform the placeable. You can also use your scroll wheel for more precise transformations. Holding shift allows for even more precise transformations. Holding alt enables snapping.

Left click or press return to confirm your transformation. Right click or press escape to discard it.

You can also enter values into the properties panel on the right if you want to be spot on.

Changing types

Certain assets have different types. For example, a goal can be blue, green or red. Simply select your goal (or goals), and select the type using the properties panel on the right.

Exporting

Exporting an LZ compressed StageDef for SMB 1 or SMB 2 is done by clicking the export button, and choosing the game version you want to modify. Make sure to export the compressed StageDef if you wish to directly patch the game. You can also export an smbcnv config and create the LZ using other tools if you wish.
